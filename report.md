# E-Commerce Sales Analysis Report

## 1. Project Title
E-Commerce Sales Analysis

---

## 2. Objective
The objective of this project is to analyze an e-commerce sales dataset, understand sales performance, identify the best-selling products, compare sales across different regions, and visualize sales trends using Python.

---

## 3. Dataset Information

The dataset contains the following columns:

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

---

## 4. Tools Used

- Python
- Pandas
- Matplotlib
- VS Code

---

## 5. Data Cleaning

The following data cleaning steps were performed:

- Removed duplicate records
- Removed missing values
- Checked dataset structure
- Verified column names and data types

---

## 6. Data Analysis

The following analyses were performed:

- Total Sales
- Average Sales
- Highest Sale
- Lowest Sale
- Product Wise Sales
- Region Wise Sales
- Daily Sales Trend

---

## 7. Visualizations

The following charts were created:

1. Product Wise Sales (Bar Chart)
2. Region Wise Sales (Pie Chart)
3. Daily Sales Trend (Line Chart)

---

## 8. Key Insights

- The project identified the highest-selling product.
- Regional sales performance was compared.
- Daily sales trends were analyzed.
- Total and average sales were calculated.
- Charts made the sales data easier to understand.
